# CodeBricks
CodeBricks is a JavaFX desktop application that allows users to log in and practice data structure concepts through interactive quizzes.

---

## Getting Started

### Prerequisites
Make sure you have installed:
- Java (JDK 17+)
- Maven (or use the included Maven Wrapper)

---

## Run the Application

### Option 1 (Recommended - Maven Wrapper)
```bash
./mvnw javafx:run
